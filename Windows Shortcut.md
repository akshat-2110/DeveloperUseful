## Windows

> **Open Terminal On Particular Folder**
- `Shift + Right Click` -> `Open command window here`.

> **Minimize/Maximize**
- `Window Key + Arrow Up` : Maximize & `Window Key + Arrow Down` : Minimize

> **Create new folder**
- `Ctrl + Shift + n`

## Browser

> **New Tab**
- `Ctrl + t`

> **Close tab/window**
- `Ctrl + w`

> **Write URL in new Tab**
- `Ctrl + l`

> **Open previously closed Tab**
- `Ctrl + Shift + t`

> **Switch between tab**
- `Ctrl + Arrow Up/Down`

## MS-Docs

> **Increase font size**
- `Ctrl + Shift + >`

> **Font style**
- Bold = `Ctrl + B`

## PDF Viewer

> **Previous View in Adobe Acrobat PDF Reader**
- `Alt + Left Arrow`
